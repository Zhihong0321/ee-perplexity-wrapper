ü™™ MYKAD & Namecard Extractor API

New Feature

Extract personal information from MYKAD cards or customer namecards with minimal JSON output.

üìÑ Features
  • Image & PDF upload
  • Minimal JSON response
  • Auto thread deletion
  • ~5-8s response time
üîó Endpoint
POST /api/extract-mykad
üìã Extracted Fields
  • name
  • mykad_id
  • address
  • contact_number

üöÄ Quick Start (curl)

curl -X POST "http://localhost:8000/api/extract-mykad" \
-F "file=@mykad_front.jpg" \
-F "account_name=yamal"

Response: JSON with extracted name, MYKAD ID, address, and contact number

üêç Python Example

import requests

with open('mykad_front.jpg', 'rb') as f:
    response = requests.post(
        'http://localhost:8000/api/extract-mykad',
        files={'file': f},
        data={'account_name': 'yamal', 'model': 'gemini-3-flash'}
    )

# Get JSON response
result = response.json()
print(f"Name: {result['data']['name']}")
print(f"MYKAD ID: {result['data']['mykad_id']}")
print(f"Address: {result['data']['address']}")
print(f"Contact: {result['data']['contact_number']}")
